Embedded Systems
MCU Architecture · Protocols · Firmware · PCB Design
How hardware and software meet. These notes cover the full stack from silicon peripherals and communication protocols up through firmware architecture and PCB layout — with emphasis on understanding what's actually happening at the register and signal level, not just how to use a HAL.
Architecture & Internals
1 · MCU Architecture ARM Cortex-M core, register file, modes & privilege, fixed memory map, reset sequence, EFR32 series overview.
2 · Memory & Clock Systems Flash wait states, SRAM layout (.data/.bss/stack/heap), clock tree, PLLs, EFR32 CMU, energy modes EM0–EM4.
3 · Interrupts & the NVIC Exception model, vector table, NVIC priority levels, preemption, tail-chaining, interrupt-safe code.
4 · DMA Why DMA exists, transfer descriptors, LDMA (EFR32), circular buffer pattern for ADC/UART streaming.
Peripherals
5 · GPIO Push-pull vs open-drain, input pull-ups, Schmitt trigger, alternate function routing, GPIO interrupts, debouncing.
6 · Timers & PWM Counter modes (up/up-down), compare/capture channels, PWM generation, dead-time insertion, LETIMER for low-power periodic wakeup.
7 · ADC & DAC SAR architecture, quantization, reference voltage, acquisition time, antialiasing filter, DMA+TIMER+PRS acquisition pipeline, PWM-as-DAC.
Communication Protocols
8 · UART Frame format (start/data/parity/stop), baud rate generation and error tolerance, RS-232 vs TTL levels, RTS/CTS flow control, interrupt-driven receive, common failure modes.
9 · SPI Full-duplex shift register operation, CPOL/CPHA clock modes, hardware vs software chip select, independent CS vs daisy-chain topologies, clock frequency limits, EFR32 USART sync mode.
10 · I2C Open-drain topology and pull-up sizing, START/STOP conditions, 7-bit and 10-bit addressing, ACK/NAK, clock stretching, Repeated START for register reads, EFR32 I2C driver, bus lockup recovery.
11 · CAN Bus Differential signalling (CAN-H/CAN-L), dominant/recessive encoding, non-destructive arbitration by message ID, data frame format, bit stuffing, error confinement state machine (active/passive/bus-off), termination.
12 · USB Fundamentals Enumeration sequence, descriptor hierarchy (device→config→interface→endpoint), four transfer types (control/bulk/interrupt/isochronous), USB-FS vs HS electrical requirements, CDC-ACM virtual serial port.
Firmware Practices
13 · Bare-Metal Patterns C startup sequence (.data/.bss init, init_array), superloop vs interrupt-driven hybrid, FSM design, interrupt-safe ring buffers, driver layering (HAL/driver/application), volatile and memory barriers (DMB/DSB/ISB).
14 · RTOS Concepts Tasks and task states, preemptive priority-based scheduling, context switch mechanics (PendSV), binary semaphores, mutexes with priority inheritance, queues, priority inversion, stack overflow detection.
15 · Debugging & Test SWD/JTAG and the ARM CoreSight debug architecture (DAP, AHB-AP, FPB, DWT, ITM), OpenOCD and GDB workflow, HardFault analysis (CFSR/MMFAR/BFAR), printf vs ITM tracing, logic analyzer protocol decoding, oscilloscope signal integrity techniques.
PCB Design
16 · Layout Fundamentals PCB layer stackup (2-layer vs 4-layer), return current path and why ground plane slots cause EMI, bulk vs local decoupling capacitor placement (100 nF ceramics at each VDD pin), via stitching, clearance/creepage/board-edge keepout rules.
17 · Mixed-Signal Layout Analog/digital floorplanning, single continuous ground plane (why split planes are wrong), AVDD/VREF bypass, ADC input trace routing, InAmp layout for biosensors, guard rings for high-impedance nodes, capacitive/inductive/common-impedance coupling and ferrite bead filtering.
18 · Power & Thermal Buck/boost hot loop layout (C_IN placement, minimise switching loop area), PDN target impedance and capacitor cascade, thermal resistance model (θ_JC + θ_CS + θ_SA), exposed pad + thermal via arrays (no thermal relief), copper pour sizing, T_J estimation at worst-case ambient.
Lesson 19 · High-Speed & RF
Impedance Control, Differential Pairs, Antennas
Controlled impedance traces (microstrip and stripline), differential pair routing and length matching, USB and Ethernet layout rules, PCB antenna design (inverted-F, chip antenna placement), RF keepouts, and matching network placement.
