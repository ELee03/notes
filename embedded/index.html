Embedded Systems
MCU Architecture · Protocols · Firmware · PCB Design
How hardware and software meet. These notes cover the full stack from silicon peripherals and communication protocols up through firmware architecture and PCB layout — with emphasis on understanding what's actually happening at the register and signal level, not just how to use a HAL.
Architecture & Internals
1 · MCU Architecture ARM Cortex-M core, register file, modes & privilege, fixed memory map, reset sequence, EFR32 series overview.
2 · Memory & Clock Systems Flash wait states, SRAM layout (.data/.bss/stack/heap), clock tree, PLLs, EFR32 CMU, energy modes EM0–EM4.
3 · Interrupts & the NVIC Exception model, vector table, NVIC priority levels, preemption, tail-chaining, interrupt-safe code.
4 · DMA Why DMA exists, transfer descriptors, LDMA (EFR32), circular buffer pattern for ADC/UART streaming.
Peripherals
Lessons 5–7
GPIO, Timers & PWM, ADC & DAC
GPIO configuration registers, alternate function mapping, open-drain vs push-pull, input pull-ups and Schmitt triggers. Timer modes (up-counter, capture/compare, centre-aligned), PWM generation and dead-time insertion. SAR ADC operation, sampling rate, reference voltage, antialiasing, and DMA-driven ADC acquisition for biosignals.
Communication Protocols
Lesson 8 · UART
Universal Asynchronous Receiver-Transmitter
Frame format (start bit, data bits, parity, stop bits), baud rate generation and clock error tolerance, RS-232 vs TTL levels, flow control (RTS/CTS), and common failure modes (framing errors, overrun).
Lesson 9 · SPI
Serial Peripheral Interface
The four CPOL/CPHA mode combinations and why they matter, full-duplex shift-register operation, chip select management (hardware vs software NSS), multi-device bus topologies, and maximum clock frequency constraints from trace capacitance.
Lesson 10 · I2C
Inter-Integrated Circuit
Open-drain bus topology and pull-up resistor sizing, START/STOP conditions, 7-bit and 10-bit addressing, ACK/NAK arbitration, clock stretching, repeated START for register reads, and common issues (bus lockup, address conflicts).
Lesson 11 · CAN Bus
Controller Area Network
Differential signalling (CAN-H/CAN-L), dominant/recessive bit encoding, non-destructive arbitration by message ID, frame format (standard and extended), bit stuffing, error detection and the error confinement state machine (active/passive/bus-off), and termination resistor placement.
Lesson 12 · USB Fundamentals
Universal Serial Bus
Device enumeration, descriptor hierarchy (device → configuration → interface → endpoint), the four transfer types (control, bulk, interrupt, isochronous), USB-FS vs USB-HS electrical requirements, and CDC-ACM as a practical virtual serial port example.
Firmware Practices
Lessons 13–15
Bare-Metal Patterns, RTOS, Debugging
Bare-metal startup sequence (vector table, stack setup, C runtime init), state machine design, ring buffers and interrupt-safe data structures, peripheral driver layering. RTOS scheduling (preemptive vs cooperative), task priorities, mutexes and binary semaphores, common RTOS pitfalls (priority inversion, stack overflow). Hardware debugging with SWD/JTAG, OpenOCD and GDB, logic analyzer and oscilloscope techniques for protocol decoding.
PCB Design
Lesson 16 · Layout Fundamentals
Stackup, Ground Planes, Decoupling
Layer stackup choices (2-layer vs 4-layer), continuous ground plane and why return current path matters, bulk and local decoupling capacitor placement and value selection, via stitching, and mechanical design rules (clearance, creepage, board edge keepouts).
Lesson 17 · Mixed-Signal Layout
Analog/Digital Partitioning
Separating analog and digital domains, ADC and DAC layout (AVDD/DVDD isolation, reference bypass, input filtering), instrumentation amplifier placement for biosensor front-ends, guarding high-impedance nodes, and managing digital noise coupling into analog circuitry.
Lesson 18 · Power & Thermal
Switching Regulators, Power Delivery, Heat
Buck and boost converter layout (tight switching loop, input capacitor placement), power delivery network (PDN) design, thermal relief in solder pads vs thermal resistance, copper pours for heat spreading, and estimating junction temperature from power dissipation.
Lesson 19 · High-Speed & RF
Impedance Control, Differential Pairs, Antennas
Controlled impedance traces (microstrip and stripline), differential pair routing and length matching, USB and Ethernet layout rules, PCB antenna design (inverted-F, chip antenna placement), RF keepouts, and matching network placement.
