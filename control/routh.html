4 · Routh-Hurwitz Stability
Characteristic Polynomial · Routh Array · LHP Roots · Stability Margin
Notation used in this lesson
$\Delta(s)$Characteristic polynomial: $\Delta(s) = 1 + L(s) = $ denominator of closed-loop TF
$r_i$$i$th element of the first column of the Routh array
$N_-$Number of sign changes in the first column = number of RHP roots
$K_{cr}$Critical (ultimate) gain: the gain at which the system is marginally stable
$\omega_{cr}$Frequency of sustained oscillation at the stability boundary
A closed-loop system is stable if and only if all roots of its characteristic polynomial $\Delta(s)$ lie in the left half of the complex plane. You could find the roots numerically — but a 6th-order polynomial has 6 roots to compute, and if you want to ask "for which values of the gain $K$ is the system stable?" you would have to repeat the computation for every $K$. The Routh-Hurwitz criterion answers the stability question directly from the polynomial coefficients, without finding any roots. It turns a root-finding problem into a table calculation, and it turns a parametric design question into an algebraic inequality.
The stability problem
Recall from Lesson 1: the closed-loop transfer function for a unity-feedback system with plant $P(s)$ and controller $C(s)$ is $$T(s) = \frac{C(s)P(s)}{1 + C(s)P(s)}$$ The system is stable if and only if all poles of $T(s)$ — equivalently, all roots of the characteristic polynomial $$\Delta(s) = 1 + C(s)P(s) = a_n s^n + a_{n-1}s^{n-1} + \cdots + a_1 s + a_0$$ — have strictly negative real parts (lie in the open left half-plane).
For a second-order polynomial $s^2 + b s + c$, the roots are in the LHP if and only if $b > 0$ and $c > 0$ — check by inspection. For third and higher order, it is not so simple. The Routh-Hurwitz criterion is the systematic answer.
Necessary conditions first
Before building the full array, apply these quick filters:
  • All coefficients must be present. If any coefficient $a_k$ is zero (for $0 \le k \le n$), the polynomial is missing a term. A zero coefficient is not sufficient for instability, but it means there are either roots on the imaginary axis or roots in the RHP. Exception: a missing $a_0$ (constant term) means there is a root at $s=0$ — a pure integrator, marginally stable at best.
  • All coefficients must have the same sign. If the coefficients alternate in sign or some are negative, there is definitely at least one RHP root. Necessary condition: $a_k > 0$ for all $k$ (or all $< 0$). This is not sufficient — a polynomial with all positive coefficients can still be unstable — but it's a fast pre-check that catches many common errors.
Example of necessary condition failure: $s^3 + 2s^2 - s + 4$ — the coefficient of $s$ is negative, so there is at least one RHP root. Stop here; no need to build the Routh array.
Building the Routh array
Given the characteristic polynomial with all positive coefficients: $$\Delta(s) = a_n s^n + a_{n-1}s^{n-1} + a_{n-2}s^{n-2} + \cdots + a_1 s + a_0$$
Step 1. Write the first two rows by alternating coefficients:
$s^n$ $a_n$ $a_{n-2}$ $a_{n-4}$ $\cdots$
$s^{n-1}$ $a_{n-1}$ $a_{n-3}$ $a_{n-5}$ $\cdots$
Step 2. Each subsequent row is computed from the two rows above it using $2\times 2$ determinants. Label the running rows as $b$, $c$, $d$, $\ldots$ The formula for the $b$ row (third row):
$$b_1 = \frac{a_{n-1}\cdot a_{n-2} - a_n \cdot a_{n-3}}{a_{n-1}}, \quad b_2 = \frac{a_{n-1}\cdot a_{n-4} - a_n \cdot a_{n-5}}{a_{n-1}}, \quad \ldots$$
More compactly: each entry is $-\frac{1}{\text{pivot}} \det\begin{pmatrix} \text{pivot} & \text{one right} \\ \text{below pivot} & \text{below right} \end{pmatrix}$ where "pivot" is the first element of the row above.
Continue until the row for $s^0$ is complete. The array always has $n+1$ rows. The last two rows each have one entry (except for the special cases below).
Concrete 4th-order template for $\Delta(s) = a_4 s^4 + a_3 s^3 + a_2 s^2 + a_1 s + a_0$:
$s^4$ $a_4$ $a_2$ $a_0$
$s^3$ $a_3$ $a_1$ $0$
$s^2$ $b_1 = \dfrac{a_3 a_2 - a_4 a_1}{a_3}$ $b_2 = \dfrac{a_3 a_0 - a_4 \cdot 0}{a_3} = a_0$ $0$
$s^1$ $c_1 = \dfrac{b_1 a_1 - a_3 b_2}{b_1}$ $0$
$s^0$ $d_1 = b_2 = a_0$
The stability test
Once the array is complete, read off the first column. This is the sequence $a_n,\, a_{n-1},\, b_1,\, c_1,\, d_1,\, \ldots$ (one entry per row).
Routh-Hurwitz Theorem. The number of roots of $\Delta(s)$ in the right half-plane equals the number of sign changes in the first column of the Routh array.

Corollary: the system is stable if and only if all first-column elements have the same sign (and are non-zero).
When all coefficients are positive and all first-column entries are positive, the polynomial has all roots in the LHP → stable. If any first-column entry is negative, count the sign changes — each sign change corresponds to one root crossing into the RHP.
Special cases
Two situations break the standard algorithm:
Case 1: Zero in the first column, but not an entire zero row. The next row's computation would divide by zero. Fix: replace the zero with a small positive number $\varepsilon > 0$, complete the array, then take the limit $\varepsilon \to 0^+$ to determine the signs. The number of sign changes tells you the number of RHP roots.
Case 2: An entire row of zeros. This happens when the polynomial has roots that are symmetric about the origin — either a purely imaginary conjugate pair $\pm j\omega$ (roots on the imaginary axis) or real roots with opposite signs (one in LHP, one in RHP).
  • Find the auxiliary polynomial $P_{\text{aux}}(s)$ from the row above the zero row. If the zero row is at the $s^k$ level, $P_{\text{aux}}(s)$ uses the coefficients of the $(s^{k+1})$ row (or row above), treating them as coefficients of $s^{k+1},\, s^{k-1},\, s^{k-3},\ldots$
  • Replace the zero row with the coefficients of $dP_{\text{aux}}(s)/ds$.
  • Continue the array. Roots of $P_{\text{aux}}(s)$ include the imaginary-axis or symmetric roots.
Physical meaning of a zero row. A zero row at $s^2$ means the characteristic polynomial has a factor $s^2 + \omega^2$ — a pair of imaginary-axis poles producing undamped oscillation. This is the boundary of stability: the system oscillates forever (neither grows nor decays). In a gain-tuning problem, this gives you the critical gain $K_{cr}$ and the oscillation frequency $\omega_{cr} = \sqrt{\omega^2}$.
Finding the stability range of the gain
The most common design application: given a closed-loop system with a free gain parameter $K$, find the range of $K$ for which the system is stable.
Method:
  1. Write out the characteristic polynomial $\Delta(s)$ with $K$ as a parameter.
  2. Build the Routh array; each entry will be a function of $K$.
  3. Require all first-column entries to be positive. This gives a set of inequalities in $K$.
  4. Solve the inequalities. The intersection is the stability range.
The critical gain $K_{cr}$ is the value at which one first-column entry transitions through zero. At $K = K_{cr}$, a row-of-zeros condition arises, and $\omega_{cr}$ can be read from the auxiliary polynomial.
Worked examples
Example 1: Third-order system. Determine stability of $\Delta(s) = s^3 + 6s^2 + 11s + 6$.
All coefficients positive → passes necessary check. Build the array:
$s^3$111
$s^2$66
$s^1$ $\dfrac{6\cdot11-1\cdot6}{6}=10$ 0
$s^0$6
First column: $1, 6, 10, 6$ — all positive, zero sign changes. System is stable. (The roots are $s = -1, -2, -3$, all in the LHP — this is the characteristic polynomial of a fully-damped third-order system.)
Example 2: An unstable polynomial. $\Delta(s) = s^3 + 2s^2 - s + 2$.
Coefficient of $s$ is $-1 < 0$ → fails the necessary condition immediately. Unstable. (No need to build the array.)
Example 3: Stability range of gain $K$. Unity-feedback system with open-loop $L(s) = K/[s(s+1)(s+3)]$. Find the range of $K > 0$ for stability.
Characteristic polynomial: $$\Delta(s) = 1 + L(s) = 1 + \frac{K}{s(s+1)(s+3)} = \frac{s(s+1)(s+3) + K}{s(s+1)(s+3)}$$ Numerator: $s(s+1)(s+3) + K = s^3 + 4s^2 + 3s + K$. All coefficients: $1, 4, 3, K$. For $K > 0$, the necessary condition is satisfied.
Build the Routh array with $K$ as a parameter:
$s^3$13
$s^2$4K
$s^1$ $\dfrac{4\cdot3-1\cdot K}{4}=\dfrac{12-K}{4}$ 0
$s^0$$K$
First-column elements: $1,\; 4,\; \dfrac{12-K}{4},\; K$. For all to be positive: $$1 > 0 \;\checkmark, \quad 4 > 0 \;\checkmark, \quad \frac{12-K}{4} > 0 \Rightarrow K < 12, \quad K > 0$$ Stable for $0 < K < 12$.
Critical gain: $K_{cr} = 12$. At this gain, the $s^1$ row is zero. The auxiliary polynomial comes from the $s^2$ row: $P_{\text{aux}}(s) = 4s^2 + 12 = 4(s^2 + 3)$. So $\omega_{cr} = \sqrt{3} \approx 1.73$ rad/s — the system oscillates at this frequency when on the stability boundary.
Example 4: Fourth-order with zero row (purely imaginary roots). $\Delta(s) = s^4 + s^3 + 3s^2 + 3s + 2$. Check for stability.
$s^4$132
$s^3$130
$s^2$$\dfrac{1\cdot3-1\cdot3}{1}=0$$\dfrac{1\cdot2-1\cdot0}{1}=2$
The $s^2$ row starts with zero — not an entire zero row, but a zero in the first column. Replace with $\varepsilon > 0$, continue:
$s^4$132
$s^3$130
$s^2$$\varepsilon$2
$s^1$ $\dfrac{3\varepsilon - 2}{\varepsilon}=3-\dfrac{2}{\varepsilon}$ 0
$s^0$2
As $\varepsilon \to 0^+$: $3 - 2/\varepsilon \to -\infty$ (negative). First column: $1,\;1,\;\varepsilon^+,\;-\infty,\;2$. Sign changes: $(+) \to (+)$, $(+) \to (+)$, $(+) \to (-)$, $(-) \to (+)$ — two sign changes. This polynomial has 2 roots in the RHP. Unstable.

Food for thought
Reflection questions — no answers given
  1. Routh-Hurwitz tells you how many roots are in the RHP but not where they are. How does this compare to the root locus (Lesson 5), which shows you exactly where the closed-loop poles move as $K$ varies? When would you use Routh-Hurwitz instead of root locus, and vice versa?
  2. The critical gain $K_{cr}$ in Example 3 is 12. If the plant gain drifts by 20% (a real physical system is not perfectly modeled), by how much does the closed-loop gain margin shrink? Express this as a percentage of $K_{cr}$. This is the practical meaning of a gain margin.
  3. For a first-order system $s + a_0$, the Routh array is trivially two entries: $a_1=1$ and $a_0$. Stable iff $a_0 > 0$. For a second-order system $s^2 + a_1 s + a_0$, the condition is $a_1 > 0$ and $a_0 > 0$. Verify these match the Routh criterion. Can you derive a general Routh-like condition for third-order systems $s^3 + a_2 s^2 + a_1 s + a_0$ directly from the table (without looking it up)?
Common pitfalls
Mistakes worth knowing about in advance
  • Applying Routh to the plant, not the closed-loop. The characteristic polynomial $\Delta(s) = 1 + C(s)P(s)$ is the denominator of the closed-loop transfer function. Applying the Routh test to $P(s)$ or $C(s)$ alone tells you about the plant's open-loop stability, not the closed-loop stability. They are different polynomials with different roots.
  • Forgetting that a zero row ≠ a zero first entry. A zero in the first column (but with other nonzero entries in that row) and an entire zero row are two completely different situations requiring different fixes. A zero first entry → use $\varepsilon$ substitution. An entire zero row → use the auxiliary polynomial from the row above. Confusing them leads to the wrong stability count.
  • Reading "all positive" and stopping too early. All positive coefficients is a necessary condition, not sufficient. $s^4 + s^3 + 4s^2 + 4s + K$ with $K > 0$ has all positive coefficients — but the Routh array reveals that it's unstable for $K > 3$ due to a sign change in the first column. Always complete the array.
  • Treating the stability range as symmetric in $K$. The stability range $0 < K < K_{cr}$ from Example 3 is bounded below by $K > 0$ (plant-model constraint) and above by $K_{cr} = 12$ (first column constraint). Different systems give different ranges — sometimes one-sided (all $K > 0$ stable), sometimes two-sided. Read the inequalities; don't assume.

The stability boundary from Routh ($K_{cr}$) corresponds to the gain where the root locus crosses the imaginary axis — they give the same answer through different means.
