The reachability question
You've built the state-space model $\dot{x} = Ax + Bu$, $y = Cx$. You want to design state feedback (Lesson 12) to place the closed-loop poles wherever you want, and an observer (Lesson 14) to estimate states you can't directly measure. But before either design is possible, two structural questions must be answered:
  • Controllability: Can the input $u(t)$ reach every state in $\mathbb{R}^n$? If some states are unreachable by $u$, no controller — however clever — can move them. State feedback cannot assign those modes.
  • Observability: Does the output $y(t)$ carry enough information to reconstruct $x(t)$? If some states are invisible to $y$, no observer can estimate them. You cannot feed back what you cannot observe.
These aren't design choices — they are properties of the plant $(A, B, C)$ itself. If the plant is not controllable or not observable, no amount of control design can compensate. The system's internal structure has fundamental limitations.
Controllability
A system $(A, B)$ is controllable if, for any initial state $x_0$ and any target state $x_f$, there exists a finite time $T$ and an input $u(t)$ defined on $[0, T]$ such that $x(0) = x_0$ and $x(T) = x_f$.
In other words: starting from anywhere, the input can drive the state to anywhere. This is a strong statement — it doesn't just ask "can we stabilize?" but "can we reach any target?"
Intuition via the state response. From Lesson 10, the state at time $T$ is: $$x(T) = e^{AT}x_0 + \int_0^T e^{A(T-\tau)}Bu(\tau)\,d\tau$$ The reachable set from $x_0 = 0$ is: $$\mathcal{R}(T) = \left\{\int_0^T e^{A(T-\tau)}Bu(\tau)\,d\tau \,:\, u \in L^2[0,T]\right\}$$ The system is controllable iff $\mathcal{R}(T) = \mathbb{R}^n$ for some $T > 0$. The columns of $e^{A\tau}B$ (as $\tau$ varies) span the reachable directions. Expanding $e^{A\tau} = I + A\tau + A^2\tau^2/2! + \cdots$, the reachable directions are spanned by $\{B,\, AB,\, A^2B,\, \ldots\}$. By Cayley-Hamilton, higher powers of $A$ are redundant beyond $A^{n-1}$. So the reachable set is spanned by the columns of:
$$\mathcal{C} = \begin{bmatrix}B & AB & A^2B & \cdots & A^{n-1}B\end{bmatrix}$$
Controllability matrix test. The system $(A, B)$ is controllable if and only if: $$\text{rank}(\mathcal{C}) = n$$ where $\mathcal{C} \in \mathbb{R}^{n \times nm}$ is the controllability matrix. For a SISO system ($m=1$), $\mathcal{C}$ is square ($n \times n$) and the condition becomes $\det(\mathcal{C}) \neq 0$.
Physical interpretation of the columns. Each column block $A^k B$ represents the direction the input can influence the state after $k$ "steps" of dynamics have propagated it. $B$ is the direction the input acts on directly. $AB$ is that direction propagated through one application of $A$ (one time step into the future). $A^2B$ is two time steps, and so on. Controllability requires all $n$ state dimensions to be reachable via some combination of these propagated input directions.
Simple uncontrollable example. Let $A$ be block-diagonal with two subsystems, and $B$ drive only the first: $$A = \begin{bmatrix}-1 & 0 \\ 0 & -2\end{bmatrix}, \quad B = \begin{bmatrix}1 \\ 0\end{bmatrix}$$ Then $AB = \begin{bmatrix}-1\\0\end{bmatrix}$, so $\mathcal{C} = \begin{bmatrix}1 & -1 \\ 0 & 0\end{bmatrix}$. The second row is all zeros: $\text{rank}(\mathcal{C}) = 1 \neq 2$. The state $x_2$ (the second subsystem) can never be influenced by $u$. It evolves freely as $x_2(t) = e^{-2t}x_2(0)$ regardless of input.
Controllability Gramian
The controllability Gramian over horizon $[0,T]$ is:
$$W_c(T) = \int_0^T e^{A\tau}BB^Te^{A^T\tau}\,d\tau$$
It is always symmetric positive semi-definite (PSD). The system is controllable if and only if $W_c(T)$ is positive definite for some (and then any) $T > 0$.
The Gramian has a beautiful energy interpretation. The minimum-energy input to drive the state from $x(0) = x_0$ to $x(T) = 0$ is:
$$u^*(t) = -B^T e^{A^T(T-t)} W_c(T)^{-1} e^{AT} x_0$$
and the minimum energy required is:
$$\|u^*\|^2 = x_0^T e^{A^TT} W_c(T)^{-1} e^{AT} x_0$$
The set of states reachable from $x_0 = 0$ with unit input energy is the ellipsoid $\{x_f \,:\, x_f^T W_c(T)^{-1} x_f \leq 1\}$. The eigenvectors of $W_c(T)$ define the principal axes of this ellipsoid — directions that are easy to reach (large eigenvalues) and hard to reach (small eigenvalues). A nearly rank-deficient Gramian means some directions are practically unreachable.
Observability
A system $(C, A)$ is observable if, for any initial state $x_0$, the state can be uniquely determined from the output $y(t)$ on $[0, T]$ for some finite $T$, assuming $u(t) = 0$ (or $u$ is known).
The output is $y(t) = Ce^{At}x_0$ (for $u=0$). Expanding: $y(t) = C\bigl(I + At + A^2t^2/2! + \cdots\bigr)x_0$. At $t=0$: $y(0) = Cx_0$. Differentiating: $\dot{y}(0) = CAx_0$, $\ddot{y}(0) = CA^2x_0$, etc. So observing $y$ and its derivatives gives the equations $[C;\, CA;\, CA^2;\, \ldots]x_0 = [\ldots]$. We can uniquely solve for $x_0$ iff the matrix on the left has full column rank:
$$\mathcal{O} = \begin{bmatrix}C \\ CA \\ CA^2 \\ \vdots \\ CA^{n-1}\end{bmatrix}$$
Observability matrix test. The system $(C, A)$ is observable if and only if: $$\text{rank}(\mathcal{O}) = n$$ where $\mathcal{O} \in \mathbb{R}^{np \times n}$ is the observability matrix. For SISO, $\mathcal{O}$ is square and the condition is $\det(\mathcal{O}) \neq 0$.
Simple unobservable example. Let the system have two modes with the same natural frequencies but connected so the output only sees their difference: $$A = \begin{bmatrix}-1 & 0 \\ 0 & -1\end{bmatrix}, \quad C = \begin{bmatrix}1 & -1\end{bmatrix}$$ $\mathcal{O} = \begin{bmatrix}1 & -1 \\ -1 & 1\end{bmatrix}$. This is rank 1: the common mode $x_1 = x_2$ is invisible to $y = x_1 - x_2 = 0$. Even though both states evolve at $e^{-t}$, the output is always zero when they start equal — no way to distinguish the common mode from the initial condition.
Duality principle
Controllability and observability are dual to each other:
$$(A, B) \text{ controllable} \iff (A^T, B^T) \text{ observable}$$ $$(C, A) \text{ observable} \iff (C^T, A^T) \text{ controllable}$$
Proof: $\text{rank}(\mathcal{C}) = \text{rank}([B,AB,\ldots,A^{n-1}B]) = \text{rank}([B^T; B^TA^T; \ldots; B^T(A^T)^{n-1}]^T) = \text{rank}(\mathcal{O}^T\text{ for dual system})$.
This duality is not just a mathematical coincidence — it has deep physical meaning. The dual system $(A^T, C^T, B^T)$ has inputs where the original system had outputs, and outputs where it had inputs. A controllability proof in one system translates directly to an observability proof in the dual. In Lesson 14, the observer design (which requires observability) will follow directly from the state-feedback design (which requires controllability) via this duality.
PBH test
The Popov-Belevitch-Hautus (PBH) test offers an eigenvalue-based perspective on controllability. It's often more informative than the rank test because it identifies which modes are uncontrollable.
PBH controllability test. $(A, B)$ is controllable if and only if for every eigenvalue $\lambda$ of $A$: $$\text{rank}\begin{bmatrix}A - \lambda I & B\end{bmatrix} = n$$ Equivalently: no left eigenvector $w$ of $A$ is orthogonal to $B$: $$w^T A = \lambda w^T \implies w^T B \neq 0$$
Why this works. If $w^TB = 0$ for some left eigenvector $w$, then differentiating $w^Tx(t)$: $\frac{d}{dt}(w^Tx) = w^T\dot{x} = w^T(Ax + Bu) = \lambda(w^Tx) + (w^TB)u = \lambda(w^Tx)$ since $w^TB = 0$. So $w^Tx(t) = e^{\lambda t}w^Tx_0$ — the projection of the state onto $w$ evolves with no influence from $u$. This is an uncontrollable mode.
PBH observability test. $(C, A)$ is observable if and only if for every eigenvalue $\lambda$ of $A$: $$\text{rank}\begin{bmatrix}A - \lambda I \\ C\end{bmatrix} = n$$ Equivalently: no right eigenvector $v$ of $A$ is in the null space of $C$: $$Av = \lambda v \implies Cv \neq 0$$
Why this works. If $Cv = 0$, then the mode $e^{\lambda t}v$ contributes nothing to the output: $y(t) = Ce^{At}x_0 = C(\cdots e^{\lambda t}v v^T x_0/\|v\|^2 + \cdots)$. The $v$-component of $x_0$ is invisible in $y(t)$ — it's an unobservable mode.
Kalman decomposition
Every LTI system can be decomposed (by a state transformation) into four decoupled subsystems according to their controllability and observability:
CO — Controllable & Observable
These states are driven by the input and visible in the output. They appear in the transfer function. State feedback can assign their eigenvalues; an observer can estimate them.
C$\bar{\text{O}}$ — Controllable, Not Observable
The input can reach these states, but the output can't see them. They don't appear in the TF (pole-zero cancellation). Pole placement can move them, but they can't be estimated.
$\bar{\text{C}}$O — Not Controllable, Observable
These states are visible in the output, but the input can't reach them. They appear in the TF (contribute to poles) but can't be controlled. If unstable, they cause internal instability even if the TF looks stable.
$\bar{\text{C}}\bar{\text{O}}$ — Neither
Completely decoupled — neither the input reaches them nor the output sees them. They're invisible to any control design and any measurement.
The transfer function $G(s) = C(sI-A)^{-1}B$ only reflects the CO subsystem. The other three subsystems are hidden modes — their eigenvalues don't appear in $G(s)$ as poles. This is the source of pole-zero cancellations and the reason for non-minimal realizations.
Minimal realization
A state-space realization $(A, B, C, D)$ of a transfer function $G(s)$ is minimal if and only if it is both controllable and observable. A minimal realization has the smallest possible number of states — the order of the realization equals the degree of the minimal polynomial of the transfer function.
Any two minimal realizations of the same transfer function are related by an invertible state transformation $T$: if $(A_1, B_1, C_1, D)$ and $(A_2, B_2, C_2, D)$ are both minimal, then $A_2 = TA_1T^{-1}$, $B_2 = TB_1$, $C_2 = C_1T^{-1}$. They describe the same dynamics in different coordinates.
Pole-zero cancellations and minimality. A non-minimal realization has one or more of:
  • Uncontrollable modes: eigenvalues of $A$ that are zeros of $C(sI-A)^{-1}$ (output can't see them) but poles of $\det(sI-A)$.
  • Unobservable modes: eigenvalues of $A$ where $B$ has no projection into the corresponding eigenspace.
In transfer function form, these appear as exact pole-zero cancellations. The cancelled pole is a hidden mode — present in the state, absent from the TF. If that hidden mode is unstable, the system is internally unstable even if the TF looks stable. This is a genuine danger in control design — cancelling an unstable pole in the TF does not mean the system is stable!

Worked examples
Example 1 — Controllability check by rank test.
$$A = \begin{bmatrix}1 & 0 \\ 0 & 2\end{bmatrix}, \quad B = \begin{bmatrix}1 \\ 3\end{bmatrix}$$ $\mathcal{C} = [B \;|\; AB] = \begin{bmatrix}1 & 1 \\ 3 & 6\end{bmatrix}$ $\det(\mathcal{C}) = 6 - 3 = 3 \neq 0$. So $\text{rank}(\mathcal{C}) = 2$ — controllable. Both modes (eigenvalues $\lambda=1$ and $\lambda=2$) are reachable from the input. Note that both eigenvalues have positive real part: the system is open-loop unstable, but stabilizable by state feedback (controllable). This is the normal use case for state feedback design.
Example 2 — Uncontrollable mode via PBH test.
$$A = \begin{bmatrix}-1 & 0 \\ 1 & -2\end{bmatrix}, \quad B = \begin{bmatrix}0 \\ 1\end{bmatrix}$$ Eigenvalues: $\lambda_1 = -1$, $\lambda_2 = -2$. PBH test for $\lambda_1 = -1$: $\begin{bmatrix}A+I & B\end{bmatrix} = \begin{bmatrix}0 & 0 & 0 \\ 1 & -1 & 1\end{bmatrix}$ Rank = 1 (first row is all zeros). So $\lambda_1 = -1$ is uncontrollable: the input cannot excite the mode $e^{-t}$. Verify via $\mathcal{C}$: $AB = \begin{bmatrix}0\\-2\end{bmatrix}$, so $\mathcal{C} = \begin{bmatrix}0 & 0 \\ 1 & -2\end{bmatrix}$, $\det(\mathcal{C}) = 0$. ✓ Physical interpretation: $B = [0;1]$ drives only $x_2$. Through $A$, the $-1$ mode in $x_1$ is only driven by $x_1$ itself (the $(1,1)$ entry of $A$) — there's no path from the input to $x_1$.
Example 3 — Cancellation in TF = hidden unobservable mode.
Consider the transfer function $G(s) = 1/(s+2)$ realized as: $$A = \begin{bmatrix}-2 & 0 \\ 0 & -5\end{bmatrix}, \quad B = \begin{bmatrix}1 \\ 1\end{bmatrix}, \quad C = \begin{bmatrix}1 & 0\end{bmatrix}$$ $G(s) = C(sI-A)^{-1}B = \frac{1}{s+2} + \frac{0}{s+5} = \frac{1}{s+2}$. Controllability: $AB = [-2; -5]$, $\mathcal{C} = \begin{bmatrix}1&-2\\1&-5\end{bmatrix}$, $\det = -3 \neq 0$. Controllable. Observability: $CA = [-2, 0]$, $\mathcal{O} = \begin{bmatrix}1&0\\-2&0\end{bmatrix}$, $\det = 0$. Not observable! The mode $e^{-5t}$ is controllable (the input excites it) but unobservable (the output can't see it). This is a non-minimal realization — the TF shows only $1/(s+2)$. Minimal realization: $A = [-2]$, $B = [1]$, $C = [1]$.
Food for thought
  1. An uncontrollable but stable mode will decay on its own regardless of input. Is an uncontrollable stable mode a problem in practice? Under what circumstances would you accept a design that leaves a stable uncontrollable mode in the system?
  2. The PBH test says that if any left eigenvector of $A$ is orthogonal to $B$, the system is uncontrollable. This means every single column of $B$ must have a nonzero projection onto every left eigenvector. What is the geometric interpretation of this condition? What does it say about where the actuators (represented by $B$) should be placed in a physical system to ensure controllability?
  3. The Kalman decomposition splits the system into CO, C$\bar{\text{O}}$, $\bar{\text{C}}$O, and $\bar{\text{C}}\bar{\text{O}}$ parts. The transfer function only shows the CO part. If the $\bar{\text{C}}$O (uncontrollable-observable) part has an unstable eigenvalue, the system is "input-output stable" (the TF is stable) but internally unstable. Why is this dangerous even if you only care about the output being bounded?
Common pitfalls
  • "Controllable" ≠ "stabilizable." A controllable system can reach any state — including states far from the origin. Stabilizability is weaker: only requires that unstable modes be controllable. A system can be stabilizable but not fully controllable (if its uncontrollable modes are stable). For pole placement, you need full controllability; for stability, stabilizability suffices.
  • Cancelling an unstable pole-zero pair does not stabilize the system. If an unstable mode is unobservable (cancels in the TF), the closed-loop TF may look stable, but the state $x(t)$ still has an unstable component that grows without bound. This can cause actuator saturation, physical damage, and eventual output divergence. Never cancel unstable poles.
  • Near-rank-deficiency is as bad as rank deficiency. A nearly-uncontrollable system (very small $\det(\mathcal{C})$) requires enormous input energy to reach poorly-controlled directions. The Gramian eigenvalue corresponding to that direction is very small, meaning the required input grows without bound as you try to reach it.
  • The controllability matrix rank test is numerically sensitive. For high-order systems, the controllability matrix $[B, AB, \ldots, A^{n-1}B]$ can be ill-conditioned even for controllable systems, because $A^k$ may involve very large or very small numbers. In practice, use the Gramian eigenvalue test or the SVD of $\mathcal{C}$ rather than $\det(\mathcal{C})$ to assess controllability.
