Reinforcement Learning
Agent · Environment · Policy · Value Functions · MDPs · Dynamic Programming · Q-Learning · Policy Gradients
Reinforcement learning is the study of how an agent learns to make decisions by interacting with an environment. Unlike supervised learning — where a teacher provides labeled examples — the agent receives only a reward signal indicating how well it is doing, and must figure out on its own which actions led to good outcomes. Unlike unsupervised learning, the goal is not to find structure in data but to maximize cumulative reward over time.
This cluster builds the subject from the ground up: starting with the RL problem itself, through the formal framework of Markov decision processes, to the core algorithms (dynamic programming, Monte Carlo, temporal difference) and finally to the modern function approximation and policy gradient methods that underpin deep RL. The primary reference is Sutton & Barto, supplemented by Broucke's course notes for connections to adaptive control and optimal control theory.
