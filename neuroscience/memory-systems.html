Memory Systems

Declarative vs Non-Declarative · Patient H.M. · Hippocampal Trisynaptic Circuit · Place Cells · Systems Consolidation · Working Memory · Amygdala

Memory is not one thing. There are multiple distinct memory systems in the brain — anatomically separate, mechanistically different, and carrying different types of information. The clarity of this distinction came from patients: Henry Molaison (H.M.), who could learn a mirror-tracing skill perfectly but had no memory of ever practicing it; E.P., an amnesic who couldn't remember walking to the bathroom but always walked there correctly. What's remembered is stored in a different system from what's known consciously. This lesson traces the neuroscience of each major memory system — where it lives, what it stores, and what breaks when it's damaged.

A Taxonomy of Memory

Memory divides first into declarative (explicit) memory — memories you can consciously access and report — and non-declarative (implicit) memory — skills, habits, and conditioned responses that influence behaviour without conscious recollection.

Declarative memory subdivides into:

  • Episodic memory — memories of specific events with their context: who, what, where, when. "My first day at university." Hippocampus-dependent; highly susceptible to forgetting; reconstructive (not a video replay).
  • Semantic memory — general factual knowledge without episodic context: "Paris is the capital of France." Initially hippocampus-dependent; gradually becomes hippocampus-independent as it is integrated into neocortical knowledge networks.

Non-declarative memory includes:

  • Procedural memory — motor and cognitive skills: riding a bike, typing. Striatum- and cerebellum-dependent; extremely resistant to forgetting once established.
  • Classical conditioning — association of a neutral stimulus with a biologically significant event. For emotional conditioning (fear): amygdala. For skeletal muscle conditioning (eyeblink): cerebellum.
  • Priming — enhanced processing of a stimulus due to prior exposure, without conscious recognition. Neocortex (perceptual cortices); preserved in dense amnesia.
  • Habituation and sensitization — the simplest forms, studied in Aplysia (Kandel). Change in synaptic release probability at sensory-motor synapses.
Patient H.M.

Henry Molaison (1926–2008; known as "H.M." during his lifetime) underwent bilateral medial temporal lobe resection in 1953 to relieve intractable epilepsy. The surgery removed most of both hippocampi, the entorhinal cortex, and the amygdala. The epilepsy was cured. The cost was devastating: H.M. developed severe anterograde amnesia — he could not form new declarative memories. Every conversation, every encounter was forgotten within minutes. He did not know the year, the president, or that he had had surgery.

What he could do is equally revealing:

  • He retained normal intelligence, language, and working memory (he could hold a conversation)
  • He learned the mirror-tracing task over three days, showing normal skill improvement — but each day he denied ever having practiced before
  • He showed intact priming effects on word stem completion tasks
  • He had significant (though not complete) retrograde amnesia for events 3 years before surgery, but earlier memories were largely preserved

H.M.'s case established three facts that shaped modern memory research: (1) the hippocampus is necessary for new declarative memory formation; (2) it is not the permanent storage site — remote memories are stored elsewhere, likely in neocortex; (3) declarative and non-declarative memory are dissociable — the hippocampus is needed for one but not the other.

The Hippocampus

The hippocampus (named for its seahorse shape) lies in the medial temporal lobe, embedded within the parahippocampal gyrus. Its trisynaptic circuit is the best-studied memory circuit in the brain:

  1. Entorhinal cortex (EC) → Dentate Gyrus (DG) via the perforant path. The EC is the gateway to the hippocampus, receiving convergent input from virtually all neocortical association areas. DG granule cells receive this highly processed, multimodal information.
  2. Dentate Gyrus → CA3 via the mossy fibres. DG granule cells excite CA3 pyramidal neurons. CA3 also receives direct perforant path input from EC. CA3 has extensive recurrent connections within itself — the CA3 autoassociative network can complete patterns (recall from partial cues).
  3. CA3 → CA1 via the Schaffer collaterals. This is the most studied synapse in neuroscience — where LTP was first described by Bliss and Lømo. CA1 receives direct input from EC (layer III) as well, allowing comparison of CA3 output with the direct EC input.

CA1 projects to the subiculum and back to EC, closing the loop. Output from subiculum/CA1 projects to mammillary bodies, anterior thalamus, prefrontal cortex, and striatum via the fornix.

The computational roles of each stage:

  • DG — pattern separation: DG uses sparse coding and competitive inhibition to produce dissimilar representations from similar inputs. Two experiences in slightly different contexts produce non-overlapping DG patterns, preventing interference.
  • CA3 — pattern completion: The autoassociative recurrent network completes degraded or partial patterns — a partial cue reinstates the full representation. This is the attractor dynamics of memory recall.
  • CA1 — mismatch detection / output: Compares EC direct input (current experience) with CA3 output (predicted/retrieved pattern). Novelty detection when they don't match.
Place Cells and the Cognitive Map

John O'Keefe (Nobel Prize 2014) discovered in 1971 that CA1 neurons fire selectively when a rat is in a specific location in its environment — place cells. Each cell has a place field, a region of physical space where it fires; outside this region, the cell is largely silent. Together, the ~300,000 hippocampal neurons tile the environment — the hippocampus encodes a cognitive map of space.

The cognitive map hypothesis: the hippocampus doesn't just store individual memories as isolated episodes — it builds a relational map of the environment in which experiences can be located. When you navigate a new city, your hippocampus builds a spatial framework; memories of where you had dinner, which street was noisy, are encoded with their spatial context within this map.

May-Britt and Edvard Moser (Nobel Prize 2014, shared with O'Keefe) discovered grid cells in the entorhinal cortex: neurons that fire at the vertices of a regular hexagonal grid that tiles the environment. The grid pattern is maintained regardless of running speed or direction, providing a metric coordinate system that underlies place cell firing. Other cell types: head-direction cells (brainstem, retrosplenial cortex) encode heading; border cells encode proximity to walls.

These discoveries transformed our understanding of hippocampal function: the hippocampus implements a spatial coordinate system for locating experiences, and episodic memory is essentially the tagging of memories with their spatiotemporal context — "what happened where and when."

Memory Consolidation

Newly formed memories are initially fragile and hippocampus-dependent. Over time, declarative memories become hippocampus-independent (as shown by H.M.'s preserved remote memories). How?

Systems consolidation (the standard model): During active waking, cortical patterns are encoded rapidly in hippocampus. During sleep — particularly slow-wave sleep — the hippocampus reactivates recent memories ("replay") and drives the neocortical areas that were active during the original encoding. Repeated hippocampal-neocortical dialogue gradually strengthens direct cortico-cortical connections, eventually making the memory accessible without hippocampal reinstantiation. The hippocampus acts as a temporary binding device and indexer; the neocortex is the permanent store.

Evidence: (1) sleep deprivation impairs declarative memory consolidation; (2) hippocampal "sharp-wave ripples" during NREM sleep are temporally correlated with neocortical slow oscillations and thalamic spindles — a coordinated three-way rhythm that gates replay; (3) optogenetic silencing of hippocampal reactivation during sleep impairs spatial memory tested the following day.

Multiple trace theory (MTT, Nadel and Moscovitch) challenges the standard model: hippocampus remains permanently involved in retrieving vivid episodic memories. Multiple traces are created each time a memory is recalled, distributed across the hippocampus and neocortex. Remote semantic memories (general facts) do become hippocampus-independent, but remote episodic memories (specific events with rich contextual detail) remain hippocampus-dependent, because their context-richness requires ongoing hippocampal binding. MTT explains why some hippocampal amnesics lose even very remote episodic memories but retain semantic knowledge from the same period.

Working Memory and the Prefrontal Cortex

Working memory is the temporary maintenance and manipulation of information in mind — the mental "scratch pad." It is limited in capacity (~7 ± 2 items, Miller 1956; more recent estimates closer to 4 chunks) and duration (seconds, without active maintenance). Working memory is not a hippocampal function — H.M. had near-normal working memory despite profound anterograde amnesia.

The prefrontal cortex (PFC), particularly the dorsolateral prefrontal cortex (DLPFC), is critical for working memory. DLPFC neurons show sustained firing during the delay period of delayed-response tasks — maintaining the remembered information in an active neural code (persistent activity) in the absence of the stimulus. This activity is driven by recurrent excitation within PFC and supported by dopaminergic modulation from the VTA (D1 receptors stabilize persistent firing; excessive dopamine destabilizes it — the inverted-U relationship between dopamine and PFC performance explains why both too little dopamine in ADHD and too much in schizophrenia impair working memory).

Baddeley's model of working memory: a central executive (PFC) coordinates two specialized buffers: the phonological loop (rehearsing verbal information, subvocal speech; left temporal/parietal) and the visuospatial sketchpad (maintaining spatial/visual information; right parietal/occipital). A fourth component, the episodic buffer, integrates information from long-term memory with online working memory content.

Emotional Memory and the Amygdala

Emotionally significant events are remembered more vividly than neutral ones — the "flashbulb memory" phenomenon. The neural mechanism: emotionally arousing events activate the amygdala (via direct sensory input and cortical input), which modulates memory encoding in the hippocampus via norepinephrine and glucocorticoid release. This is a form of memory "priority tagging" — experiences that were emotionally significant are flagged for preferential consolidation.

The amygdala is also the primary substrate for fear conditioning: after pairing a tone (CS) with a footshock (US), the tone alone comes to elicit fear responses (CR). The basolateral amygdala (BLA) is where the CS-US association is formed; the central amygdala drives the autonomic (heart rate, blood pressure) and behavioural (freezing) fear responses via brainstem and hypothalamic projections.

Fear memories are difficult to extinguish — exposure to the CS alone gradually produces extinction learning (new CS→safety learning in prefrontal cortex), but the original fear memory is not erased; it is inhibited. The fear memory can return spontaneously, be reinstated by stress, or reappear in a new context (renewal effect). This is why exposure therapy for PTSD works but has limited generalization — extinction is context-specific.

Reconsolidation: Retrieved memories return to a labile state and must be reconsolidated. If protein synthesis is blocked during retrieval, the reactivated memory is erased. This has therapeutic implications: by pairing recall with a reconsolidation-disrupting drug or intervention, it may be possible to permanently modify fear memories. Clinical trials are ongoing (propranolol to block β-adrenergic reconsolidation of PTSD memories).

Worked Examples
Example 1 — Dissociating memory systems
Patient A has complete bilateral hippocampal damage. Patient B has complete bilateral striatal damage. Predict each patient's performance on: (a) learning a new word-pair list, (b) learning a mirror-tracing task, (c) fear conditioning to a tone.

Patient A (hippocampal damage): (a) Severely impaired — declarative episodic memory requires hippocampus. (b) Normal — procedural motor learning is striatum/cerebellum-dependent. (c) Normal acquisition — fear conditioning is amygdala-dependent; extinction learning may be impaired due to hippocampal role in context memory.

Patient B (striatal damage): (a) Normal — word-pair learning is hippocampal, not striatal. (b) Severely impaired — mirror-tracing is procedural, requires striatum. (c) Normal acquisition — fear conditioning is amygdala-dependent; habit-based avoidance behaviours may be impaired.

These double dissociations confirm the multiple memory systems hypothesis: different neural systems store different types of memory, and damage to one system selectively impairs its specific memory type while sparing others.

Food for Thought
  • The hippocampus builds a cognitive map of space via place cells. But the hippocampus is also essential for episodic memory of non-spatial events. Is there a unified theory that explains both roles?
    The "relational memory" theory (Eichenbaum) proposes that the hippocampus binds arbitrary associations between items, regardless of whether those items are spatial or not. Place fields are a special case where the items being associated are spatial locations and their sensory features. Episodic memory is another case: the hippocampus binds the "what," "where," and "when" components of an event into a single retrievable pattern. The underlying mechanism — pattern completion in a recurrent network — is the same in both cases. The cognitive map is just the most visible instance of the hippocampus's general role as a relational binder.
  • Memory recall is reconstructive — we don't play back a recording, we reconstruct from fragments, and the reconstruction can incorporate new information. What circuit-level mechanism makes memory reconstruction (rather than replay) inevitable, and what are the consequences for reliability?
    When a memory is recalled, the hippocampus reinstates the cortical pattern via pattern completion from partial cues (CA3 autoassociation). The reinstated pattern is then processed by the cortex just like a current perception. If the current context contains new related information, it gets incorporated into the reconstruction — post-event misinformation effects. Furthermore, each recall reactivates the memory (opens the reconsolidation window), and protein synthesis during this window is needed to re-stabilize it; any modification that occurs during reconsolidation is incorporated. This is the mechanistic basis for false memories: the cortex's pattern completion fills gaps with plausible (contextually consistent) information rather than veridical details.
Common Pitfalls
  • Thinking the hippocampus stores memories permanently. The hippocampus is a temporary buffer and index. Long-term memories are gradually transferred to neocortex (standard model) or maintained via distributed hippocampal-neocortical traces (MTT). This is why hippocampal damage causes anterograde amnesia (no new memories) and retrograde amnesia for recent but not very remote events.
  • Confusing episodic and semantic memory in terms of hippocampal dependence. Both initially require hippocampus, but semantic memory becomes hippocampus-independent over time. Dense amnesics can still have excellent semantic knowledge (they know Paris is in France) but no episodic recall of where or when they learned it.
  • Treating working memory and short-term memory as synonymous. Short-term memory usually refers to passive, limited-capacity storage (digit span). Working memory involves active manipulation and maintenance (n-back tasks, mental arithmetic), and depends on PFC. The two overlap but are not the same.
Connects to

Synaptic Plasticity (Lesson 7) — LTP in the hippocampal trisynaptic circuit is the cellular mechanism of memory encoding. Systems consolidation depends on LTP at hippocampal-neocortical and neocortical-neocortical synapses during sleep replay.

Sleep (Lesson 9) — hippocampal replay during sharp-wave ripples in NREM sleep is the primary mechanism of systems consolidation. Sleep spindles coordinate hippocampal-neocortical dialogue. Sleep deprivation specifically impairs hippocampus-dependent declarative memory, not procedural memory.

Reinforcement Learning (RL cluster) — the hippocampus contributes to model-based RL: storing the cognitive map and episodic memory supports planning by mentally simulating possible futures (prospective memory, "mental time travel"). The striatum supports model-free habit learning.
